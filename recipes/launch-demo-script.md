# Write a demo video script + storyboard

> You need a tight demo video for launch and want a script and shot-by-shot storyboard.

| | |
|---|---|
| **ID** | `launch-demo-script` |
| **Category** | `launch-product` |
| **Best with** | Codex |

## Prompt

```text
Write a {{LENGTH e.g. 90-second}} demo video script + storyboard for: {{PRODUCT}}.

1. Open with a 5-second hook stating the problem in the viewer's words.
2. Script the narration in short, spoken-style lines with timestamps.
3. For each line, give a matching on-screen action/shot (the storyboard column): what's shown, click path, and on-screen text.
4. Show the core 'aha' moment within the first 30 seconds.
5. End with a clear CTA and where to go next.
6. Keep it real — demo actual features, no vaporware.

Output as a two-column script/storyboard table.
```

## Safety notes

Demo only real, working features — no mockups presented as live product. Don't show real customer data or secrets on screen; use demo data. Keep timing realistic for the stated length.

## Expected output

A two-column script + storyboard: timestamped spoken narration paired with on-screen actions/shots, the aha moment in the first 30s, and a closing CTA — covering only real features.
